如何获取 API Key 和 User ID

为了在 App 中正常使用相关功能，您需要从您使用的网站获取 API 凭证。
请按照以下步骤操作：

操作步骤
 * 登录账号：在浏览器中访问 你要设置的gelbooru网站 并登录您的账户。
 * 进入设置：点击页面顶部的 Settings（设置），然后选择 Options（选项）选项卡。
 * 访问 API 凭证：在页面中找到 API Access Credentials 栏目。
 * 生成密钥：如果当前没有显示的密钥，请点击 Generate New Key 按钮。
 * 提取信息：你会得到类似如下格式的一段文本：
   &api_key=your-api-key&user_id=123456

填写指南

请将获取到的信息分别填入 App 的设置项中：
 * API Key: 填写 api_key= 后面的一串字符（例如：your-api-key）。
 * User ID: 填写 user_id= 之后的纯数字部分（例如：123456）。

> 安全提示：API Key 属于私密凭证。请妥善保管，不要将其分享给他人。
