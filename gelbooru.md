How to Retrieve Your API Key and User ID

To enable full functionality, you need to retrieve your API credentials from the site. 

Please follow these steps:

Step-by-Step Instructions
 * Sign In: Open the Gelbooru site you are configuring in your browser and log in to your account.
 * Go to Settings: Click on Settings in the navigation bar, then select the Options tab.
 * Find API Credentials: Locate the section labeled API Access Credentials.
 * Generate Key: If no key is currently active, click the Generate New Key button.
 * Identify Your Info: You will see a string of text similar to this:
   &api_key=your-api-key&user_id=123456

How to Fill in the App

Copy the specific values from the text above and paste them into the corresponding fields in the app:

| Field | Value to Copy | Example |
|---|---|---|
| API Key | The characters after api_key= | your-api-key |
| User ID | The numeric digits after user_id= | 123456 |

> Security Note: Your API Key functions like a secondary password. Keep it private and do not share it with others.
